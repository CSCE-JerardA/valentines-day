# Valentines Day Project🩷

## Live Link 
You can view the interactive site here: [https://CSCE-JerardA.github.io/valentines-day/](https://CSCE-JerardA.github.io/valentines-day/)

* I made this interactive Valentine's Day web application with CSS, HTML, and Javascript.
It forces a yes out of whoever interacts with the web page.

## Features

* The Runaway Button: The "Nah" button uses a custom teleport script. The moment the cursor gets close, it jumps to a random coordinate on the screen.

* Angry King Emote: If the user tries to click "Nah" more than 5 times, the king lets out a "GRRR"

* Sanji Window: A custom popup of Sanji that appears after 10 seconds of inactivity to remind the user that time is running out. 

* Victory Celebration: Upon clicking "YES!", the app triggers a heart-shaped confetti explosion using the canvas-confetti library and switches to a Sanji GIF


## Project Structure

* index.html: The core layout and styling of the application
* rendered.js: Handles button movement, timers, and animations
* /assets: Contains all the GIFS and images used.


## Running Locally

* Clone or download this repository to your machine.

* Ensure all the GIFS and images remain inside the /assets folder.

* Open index.html in any modern web browser.

* I would recommend Live Server extension in VS Code for the best experience
